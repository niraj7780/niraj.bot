# 🤖 Niraj File Converter Bot

A powerful Telegram bot that allows users to convert and process files easily 🚀

---

## ✅ Features

✅ Image → PDF  
✅ JPG → PNG Converter  
✅ Image Compression  
✅ Simple & Interactive Button UI  
✅ Fast Response  
✅ Runs 24/7 on Render  

---

## 🧠 How It Works

1. Start the bot with `/start`
2. Choose option using buttons:
   - 📂 Image → PDF
   - 🖼 JPG → PNG
   - 📊 Compress Image
3. Send image 📷
4. Bot processes and returns result ✅

---

## 🎯 Tech Stack

- Python 🐍
- python-telegram-bot
- Pillow (Image Processing)
- Render (Cloud Hosting)
